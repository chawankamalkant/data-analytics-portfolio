## ESG Sustainability Performance Dashboard (Power BI)

## Project Overview
This project focuses on analyzing ESG (Environmental, Social, Governance) data to evaluate sustainability performance across various companies and sectors. The dashboard provides actionable insights into key ESG metrics, helping in data-driven decision-making.

---

## Objectives
- Analyze ESG performance and key sustainability metrics  
- Track trends and sector-wise comparisons  
- Identify high and low-performing areas  

## Tools & Technologies
Power BI, Excel, DAX, Power Query  

## Key Features
- Interactive 3-page dashboard  
- KPI tracking and trend analysis  
- Dynamic filtering and sector comparison  

## Impact
- Reduced manual analysis by ~40%  
- Improved reporting efficiency and decision-making  

## Usage
Download the `.pbix` file and open it in Power BI Desktop to explore the dashboard.
