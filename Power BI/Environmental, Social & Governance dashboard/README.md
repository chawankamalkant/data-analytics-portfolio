# 📊 ESG Sustainability Performance Dashboard (Power BI)

## 📌 Project Overview
This project focuses on analyzing ESG (Environmental, Social, Governance) data to evaluate sustainability performance across various companies and sectors. The dashboard provides actionable insights into key ESG metrics, helping in data-driven decision-making.

---

## 🎯 Objectives
- Analyze ESG performance across industries
- Track key sustainability KPIs and trends
- Identify high and low-performing sectors
- Enable interactive and efficient reporting

---

## 🛠️ Tools & Technologies
- Power BI
- Microsoft Excel
- DAX (Data Analysis Expressions)
- Power Query (Data Transformation)

---

## 📊 Dashboard Features
- Interactive 3-page dashboard
- KPI Cards for ESG performance tracking
- Trend Analysis (Year-wise ESG scores)
- Sector-wise comparison
- Dynamic filters (slicers) for user interaction

---

## 🔍 Key Insights
- Identified top-performing sectors based on ESG scores
- Highlighted areas with poor sustainability performance
- Enabled faster and more efficient decision-making

---

## 📈 Impact
- Reduced manual analysis effort by ~40%
- Improved reporting efficiency and data accessibility
- Delivered actionable insights for sustainability strategies

---

## 📂 Dataset
- ESG Sustainability Performance dataset (Excel-based)
- Includes environmental, social, and governance metrics

---

## 🚀 How to Use
1. Download the `.pbix` file from this repository
2. Open it in Power BI Desktop
3. Interact with filters and visuals to explore insights
