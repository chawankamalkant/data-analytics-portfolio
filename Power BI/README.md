# Power BI Projects

This repository contains a collection of Power BI dashboards and reports showcasing data analysis, data modeling, and business intelligence skills. Each project includes clean data transformations, DAX measures, interactive visuals, and insights designed for real-world business scenarios.

## Key Features

* Comprehensive dashboards with interactive filtering
* Data cleaning and modeling using Power Query
* Advanced DAX measures for analytical calculations
* Visual storytelling through charts, KPIs, and custom visuals
* End-to-end BI workflow: data import → transformation → modeling → visualization

## Repository Structure

* **/Dashboards** – PBIX files of completed reports
* **/Data** – Sample datasets used (if included)
* **/Documentation** – Notes, screenshots, and project explanation

## Tools Used

* Power BI Desktop
* Power Query
* DAX
* Excel/CSV data sources

## Purpose

To demonstrate practical Power BI skills in business analytics, reporting, and decision-support dashboards.
