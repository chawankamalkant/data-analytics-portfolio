# K&K Biscuit Sales Dashboard

## Overview

This repository hosts the Excel project for for **K&K Biscuits**. The file is a comprehensive Microsoft Excel Macro-Enabled Workbook designed for data analysis, reporting, and business intelligence exercises.

## Contents & Features

The macro-enabled workbook contains structured data tables, along with built-in analytical components:

* **Practice Dataset:** A structured collection of data for analysis (e.g., sales, production, or customer metrics).
* **Embedded Analysis:** Includes multiple worksheets, Pivot Tables, and charts for immediate data exploration and visualization.
* **VBA Macros:** Contains embedded VBA code (`.xlsm` format) for custom functionality, automated calculations, or enhanced reporting.

## Requirements

To utilize the full functionality, including the embedded macros, the file should be opened with:

* Microsoft Excel (or compatible software that supports `.xlsm` files).

## Usage

This file is ideal for:

* Data analysts performing case studies or practicing analytical techniques.
* Business stakeholders reviewing data-driven insights.
* Users wishing to inspect or modify the underlying VBA logic.
